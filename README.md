# start-utils
## Usage

```gradle

repositories {
    maven {
        url "https://dl.bintray.com/dreamkas/start-utils"
    }
    mavenCentral()
}

dependencies {
    compile group: 'ru.dreamkas.utils', name: 'versions-utils', version: 'latest.release'
    compile group: 'ru.dreamkas.utils', name: 'time-utils', version: 'latest.release'
    compile group: 'ru.dreamkas.utils', name: 'concurrent-utils', version: 'latest.release'
}

```
